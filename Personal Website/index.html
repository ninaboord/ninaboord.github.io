Cheapest Airplane Ticket Algorithm

Project in Python | Code (In Progress)
Currently developing an algorithm that takes in a minimum number of hours of travel and spits out the cheapest possible ticket combination from all possible plane tickets. This algorithm is different from websites like Expedia because it strings together combinations of tickets that come from different airlines. For example, if you were trying to get from San Francisco to London, it might be cheaper to buy a separate plane ticket from SF to NYC and then another plane ticket from NYC to London. There are many instances of "flight hacking" like this that websites such as Expedia don't advertise.
